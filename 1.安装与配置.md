# 常用命令

* go version: 查看当前go的版本
* go env: 查看当前go的环境配置

# 国内镜像配置

## 国内镜像配置:
参考: https://github.com/goproxy/goproxy.cn#usage
```shell
$ go env -w GO111MODULE=on
$ go env -w GOPROXY=https://goproxy.cn,direct
```

## 安装goimports:
```shell
$ go get golang.org/x/tools/cmd/goimports
```

## 生成mod文件: 
```shell
$ go mod init xxx
```
